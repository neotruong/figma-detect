⚡ Pix Diff — n8n Automation Workflow

End-to-end flow from Figma version check → pixel diff → Slack notification

1
🎨
Fetch Figma Version History
Figma REST API · triggered on schedule or webhook
n8n
Call GET /v1/files/:file_id/versions to get version list
Compare latest version ID against the last stored version ID
🔀
New version detected?
✅ Yes → continue
🛑 No → stop workflow
2
📐
Size Pre-check
Quick dimension compare before heavy pixel diff
server
n8n sends current image metadata (width, height) from Figma API response
Server compares against stored before.png dimensions
Returns { sizeChanged: bool, widthDiff, heightDiff }
POST /api/size-check/:caseName lightweight, no image processing
🔀
Size changed significantly?
⚠️ Yes → flag as major, skip to report
✅ No → continue to pixel diff
3
⬇️
Fetch Newest Image from Figma
Download latest rendered PNG for each changed frame
n8n
Call GET /v1/images/:file_id?ids=:node_ids&format=png
Download the returned image URL as binary
Send the image binary to the server as after.png for the matching case_* folder
POST /api/update-after/:caseName saves new image as after.png
4
🔍
Run Pixel Diff
Full pixelmatch comparison on all updated cases
server
pixelmatch compares RGBA pixel-by-pixel · threshold 0.1
Generates magenta diff overlay image
Returns percentage + severity classification
POST /api/scan or /api/scan/:caseName for single case
🔴 Major ≥5% 🟡 Minor 0.1–5% 🟢 No Change <0.1%
5
📊
Render HTML Report
Self-contained interactive report with all diff images
server
Server builds report.html with base64-embedded images (no external deps)
Filter by severity · tap images to open lightbox
n8n receives the HTML or a hosted URL to attach/link
GET /api/report returns rendered HTML
6
💬
Send Slack Notification
n8n Slack node · summary + report link
n8n
Message includes: total cases · 🔴 major count · 🟡 minor count · 🟢 no change count
Attach report URL or upload report.html as file
Only send if at least one major or minor diff found (configurable in n8n IF node)
📡 Server API Endpoints (for n8n to call)
Method Endpoint Purpose Step
POST /api/size-check/:caseName Quick dimension compare 2
POST /api/update-after/:caseName Save new Figma image as after.png 3
POST /api/scan Full pixel diff all cases → JSON 4
GET /api/report Return rendered HTML report 5
